HTML & CSS Summery with JuNaEiD

HTML Summery

HTML (Hyper Text Markup Language) is used to define our page content, structure and meaning. You don't use it for styling purpose.Use CSS for that instead

  1. Html used 'element' to describe (annotate) content.
  2. Html elements typically have an opening tag, content and than a closing tag.
  3. You can also have void (empty) elements like images.
  4. You can also configure elements with attributes.
  5. There's a long list of available elements but you'll gain experience over time, no worries.
  6. Learn more about all available HTML elements on the MDN HTML elements reference.

CSS Summery

CSS (Cascading Style Sheet) is used for styling your page content.

  1. Styles are assigned via property-value pairs.
  2. You can assign styles via styles attribute.
  3. To avoid code duplication, you typically use global styles (e.g. via the 'style' element) instead.
  4. Alternatively,you can work with external stylesheet files which you "link" to
  5. When working with CSS,concepts like 'inheritance','specificity' and the box model should be understood.

Learn more about all available CSS properties and value on the MDN CSS property reference.
